# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-04

### Added

- Initial release of `electron-forge-publisher-azure`
- `PublisherAzure` class implementing the Electron Forge publisher interface
- Upload artifacts to Azure Blob Storage with configurable container and path prefix
- Support for SAS token and connection string authentication
- Configurable public access URL for uploaded artifacts
